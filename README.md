# 💸 'Who wants to be a millionaire?' GUI project

### This project showcases OOP, DSA, Algorithms, File Handling, Event Driven Development and many more.
- 🖥️ This is my first ever project creating a game with a desktop GUI.

## 🛠️ Built With

* **Language:** Java 25
* **Framework:** JavaFX
* **Build Tool:** Maven
* **Styling:** CSS
* **Data Storage:** Flat File Database (`.txt` file handling)

## ▶️ How to run
* **Enter command:** mvn javafx:run

## 🚀 Key Features

* **Dynamic Question Shuffling**
* **Smart UI Input Validation**
* **Smart Audience Votes implementation**
* **Persistent Leaderboard**
* **Responsive Layouts**

- I have already built a [similar project using C#](https://github.com/nemethszabii/who-wants-to-be-a-millionaire) but that one was my first ever programming project so it's obviously very 'raw'.
- 🔨 With this project I wanted to create a better version of that one. 
- 👨‍🎓 I have **learned** *Swing* and *JavaFX* basics while developing this mini game.

## 📋 Game Rules
- If you answer all questions correctly you win.
- If you answer one question incorrectly you lose.
- You have 3 helps, all of them are only available once per play.
- In case of win, your result will be written to the leaderboard with your play time.
